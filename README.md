
# Task-Tracker
### https://roadmap.sh/projects/task-tracker
# What it does?
#### The Task-Tracker is an application to organize and store your tasks, listing all the tasks by status, which will show if the task is *pending*, *completed* or *in progress*
# How to use?
#### 1 - To use the Task-Tracker you need to clone the git repository
``git clone git@github.com:RN08128/Task-Tracker.git``
#### 2 - Run the *Main* class located in src folder
#### 3 - After this, you can start creating and to do any task
